rspec-website
=============

Source for rspec.info.

Requires a recent version of Ruby (2.1.x), bundler and imagemagick (to generate favicons).

## Setup

* `brew install imagemagick`
* `bundle install`
* `middleman build`

## Developing

* `middleman server`

## Deploying

* `middleman deploy`
